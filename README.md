# QOTD-service
Qoute of the day service built with Go


#### We want to build a quote of the day service
- - -
### Dependencies
1. Go
2. gRPC
3. protoc
4. buf


### Build & Run

```
start server
`go run qotd.go --addr="127.0.0.1:3000"`
# if `addr` is left blank, it will default to `127.0.0.1:80`
```
